Эта программа показывает страны, где чаще всего встречается введённое имя

О проекте
Это приложение разработано Мыльцевым Олегом 04-08-2023 на базе API nationalize.io. Принимая имя от пользователя, оно делает запрос к api и получает JSON обьект со странами, в которых чаще всего встречается введённое имя. Затем, программа декодирует его и выводит на страницу.
